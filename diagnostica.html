Diagnostica WILLY.AI

Problemi comuni con pagina bianca

Se stai visualizzando una pagina bianca nell'applicazione WILLY.AI, potrebbe essere dovuto a uno dei seguenti problemi:

  • Errori JavaScript che impediscono il rendering dell'applicazione
  • Problemi con le dipendenze React o Material-UI
  • Errori nei componenti principali come App.tsx o Layout
  • Problemi con il sistema di routing
  • Errori nel sistema di gestione dello stato (Zustand)

Passaggi per la risoluzione

1. Verifica la console del browser

Il primo passo è controllare la console del browser per eventuali errori JavaScript:

  1. Apri l'applicazione nel browser
  2. Premi F12 o tasto destro → Ispeziona
  3. Vai alla scheda "Console"
  4. Cerca messaggi di errore in rosso

Soluzione comune: Se vedi errori relativi a moduli non trovati, esegui npm install per reinstallare le dipendenze.

2. Verifica il file index.html

Assicurati che il file index.html contenga un elemento con id "root":

Il file index.html dovrebbe contenere: <div id="root"></div>

3. Verifica i file principali

Controlla i seguenti file per errori di sintassi o problemi:

  • src/main.tsx - Punto di ingresso dell'applicazione
  • src/App.tsx - Componente principale
  • src/routes/index.tsx - Sistema di routing
  • src/components/Layout/index.tsx - Layout principale

Soluzione: Utilizza il file main-debug.tsx creato per diagnosticare problemi di rendering.

4. Problemi con Zustand

Se l'applicazione utilizza Zustand per la gestione dello stato, verifica:

  • Errori nel file src/stores/themeStore.ts
  • Problemi con l'inizializzazione degli store

Soluzione: Prova a commentare temporaneamente l'uso degli store per verificare se sono la causa del problema.

Utilizzo degli strumenti di diagnostica

Sono stati creati alcuni strumenti di diagnostica per aiutarti a risolvere il problema:

1. File debug.html

Apri il file debug.html nel browser per eseguire test diagnostici di base.

2. File debug.js

Importa e utilizza le funzioni dal file src/debug.js per diagnosticare problemi specifici.

3. File main-debug.tsx

Rinomina temporaneamente main.tsx in main.tsx.bak e rinomina main-debug.tsx in main.tsx, quindi riavvia l'applicazione per utilizzare la versione con gestione degli errori avanzata.

Soluzioni rapide

Riavvia il server di sviluppo

Spesso, riavviare il server di sviluppo può risolvere problemi temporanei:

  1. Termina il processo del server (CTRL+C nel terminale)
  2. Esegui npm run dev per riavviare

Cancella la cache del browser

La cache del browser può causare problemi con le applicazioni React:

  1. Apri le impostazioni del browser
  2. Vai alla sezione per cancellare i dati di navigazione
  3. Seleziona "Cache" e cancella

Reinstalla le dipendenze

Problemi con le dipendenze possono causare pagine bianche:

  1. Elimina la cartella node_modules
  2. Elimina il file package-lock.json
  3. Esegui npm install
  4. Riavvia con npm run dev
